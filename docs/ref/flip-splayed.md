# `!` Flip Splayed or Partitioned




```syntax
x!y    ![x;y]
```

Where `x` is a symbol list and `y` is

-   an **hsym symbol atom** denoting a **splayed** table
-   a **non-hsym symbol atom** denoting a **partitioned** table

returns the flip of `y`.
