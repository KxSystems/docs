# `!` Enumeration



_Enumerated symbol list_

Syntax: `x!y`, `![x;y]`

Where 

-   `x` is a handle to a symbol list
-   `y` is an int vector in the domain `til count x`

returns an enumerated symbol list.
Enumeration is a uniform function.

```q
q)x:`a`b`c`d
q)`x!1 2 3
`x$`b`c`d
```
